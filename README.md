# Simple-Calculator
Scientific Calculator Mini Project
A responsive scientific calculator built using HTML, CSS, and JavaScript. This project allows users to perform both basic and scientific calculations, such as sin, cos, tan, and square root. Great for beginners to practice DOM manipulation and styling.

Features
Basic arithmetic operations: addition, subtraction, multiplication, division, decimal point, clear, backspace.

Scientific operations: sin, cos, tan, and square root.

Responsive and attractive user interface.

Works on both desktop and mobile devices.

Usage
Clone or download the repository.

Open index.html in your web browser.

Start calculating!

File Structure
index.html — Calculator structure.

style.css — Styles for layout and responsiveness.

script.js — Functional logic for all operations and button clicks.

How It Works
Enter numbers and operators using the calculator buttons.

Click = to evaluate the expression.

Use sin, cos, tan, and √ for scientific calculations.

Use AC to clear, D to delete one character.
